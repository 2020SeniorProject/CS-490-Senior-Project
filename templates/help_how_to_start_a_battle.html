How to Start a Battle

This document details how to start a battle. You need to have opened a room as a Dungeon Master in order to start a battle.

  1. Upon opening a room, you will see the battlemap screen.
  2. Add at least one character or NPC to the battle map. (See: How to Add a Character to the Battlemap or How to Add a NPC to the Battlemap)
  3. Click the blue button that says "Start Battle!" located in the top right corner of the screen under the label "DM's Tools"
  4. You've started a battle!
