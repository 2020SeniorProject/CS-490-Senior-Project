How to Edit a Room

This document details how to edit a room.You need to be logged in in order to have access to rooms.

  1. From the home screen, click the "Room" tab on the navigation bar at the top.
  2. You should see a list of all your rooms. Click the name of the room you want to edit.
  3. You should a battle map with a form in the upper right corner. Make the changes you want, and click "Save Changes". If your form has errors, fix the indicated issues, then resubmit."*
  4. Upon submitting, you should be brought back to the main room page. You will see your edited room in your list of rooms!

*The form has the following rules

  • Room Name: The name can only be up to 42 characters long. You must enter something for the name.
  • DM Notes: This is optional. No restrictions on this.
  • Map URL: it must link to an image hosted on a publically available address. The address must end with .jpg, .jpeg, .webp, .gif, or .png.
