How to Add a Character to the Battlemap

This document details how to add a character to the battlemap. You must be in a room in order to add a character to the battlemap.

  1. Upon opening a room, you will see the battlemap screen.
  2. First, select the character that you want to add to the battlemap from the dropdown in the bottom right of the screen under the label "Add a Character".
  3. Click the button labeled "Add Character" just below the dropdown.
  4. You will see your character icon appear on the battlemap.
  5. You have just added a character to the battlemap!
